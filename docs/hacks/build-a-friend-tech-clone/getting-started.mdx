In this hack, developers will create a decentralized social network similar to Friend.tech using the Stacks Blockchain. This platform will allow users to buy and sell `keys` linked to their accounts, which offer access to private in-app chatrooms and exclusive content.

### Prerequisites

- Basic understanding of blockchain technology
- Familiarity with JavaScript and React for front-end development
- Basic knowledge of Clarity for smart contract development

### Tools

- [Clarinet](../../clarinet/overview)
- [Stacks.js Library](../../stacks.js/overview)
- [Leather Wallet for message signing](../../stacks.js/guides/sign-messages)

### What will we cover?

- **[Create the contract](./create-the-contracts)**: Write a Clarity smart contract that allows for keys management, and the tracking of ownership of those keys.
- **[Build the frontend](./build-the-frontend)**: Create a mechanism for users to access private chatrooms when they own a key. This could involve checking the ownership of a key when a user tries to enter a chatroom.
- **[Implement Message Signing](./handle-the-messaging)**: Allow users to sign messages with their private keys to prove ownership and access to private chatrooms.

Each section has a few challenges for you to complete. You are free to complete as many or as few of these challenges as you like. We will also provide a few ideas for you to extend your project beyond the scope of this hack.
