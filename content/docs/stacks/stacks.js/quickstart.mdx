## You will learn how to:

- Install `clarinet`
- Generate a new project with the `clarinet` CLI 
- Create a smart with Clarity
- Verify and debug your code in your terminal


          </Tab>

  <Step>
    <div className="flex flex-col">

        <p>Now let's create a `hello-world` project. This will scaffold all the necessary files and directories for us to get started.</p>
      </div>
      <div className="w-full">
        ```bash title="Terminal"
        clarinet new hello-world
        ```
      </div>
    </div>
  </Step>

  <Step>
    <div className="flex flex-col">
      <div className="w-full">
        <p className='text-xl font-bold m-0'>Create a <code className='text-lg'>say-hello</code> contract</p>
        <p>Now that our project has been generated we can start creating our first contract. Let's name this one `say-hello`.</p>
      </div>
      <div className="w-full">
        ```bash title="Terminal"
        cd hello-world
        clarinet contract new say-hello
        ```
      </div>
    </div>
  </Step>

  <Step>
    <div className="flex flex-col">
      <div className="w-full">
        <p className='text-xl font-bold m-0'>Create a <code className='text-lg'>read-only</code> function called <code className='text-lg'>say-hi</code></p>
        <p>Now that we have our `say-hello.clar` file generated, let's create a `read-only` function that prints out **"Hello World"**.</p>
      </div>
      <div className="w-full">
        ```clarity title="say-hello.clar"
        (define-read-only (say-hi)
          (print "Hello World")
        )
        ```
      </div>
    </div>
  </Step>

  <Step>
    <div className="flex flex-col">
