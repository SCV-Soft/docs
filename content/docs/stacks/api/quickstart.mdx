### You will learn the following:

- Generate a new project with the `clarinet` CLI 
- Create a smart contract with Clarity
- Verify and debug your smart contract in your terminal

---

<div className='steps space-y-10'>
  <Step>
    <div className="flex flex-col space-y-3">
      <div className="w-full">
        <p className='text-lg font-bold m-0 dark:text-primary-foreground'>Install <code className='text-lg'>clarinet</code></p>
        <div className='flex flex-col space-y-0'>
          <p>Hiro's [Clarinet](/stacks/clarinet/installation) package comes with several helpful commands to get you up and running with your project.</p>
          <p>To get started, install the package on your machine:</p>
        </div>
      </div>
      <div className="w-full">
        <Tabs items={['macOS', 'Windows']}>
          <Tab value="macOS">
          ```bash title="Terminal"
          brew install clarinet
          ```
          </Tab>
          <Tab value="Windows">
          ```bash title="Terminal"
          winget install clarinet
          ```
          </Tab>
        </Tabs>
      </div>
    </div>
  </Step>

  <Step>
    <div className="flex flex-col space-y-3">
      <div className="w-full">
        <p className='text-lg font-bold m-0 dark:text-primary-foreground'>Generate a new <code className='text-lg'>hello-world</code> project</p>
        <div className='flex flex-col space-y-0'>
          <p>Navigate to a directory in your terminal where you want to create your project. Then in your terminal, run the following command:</p>
        </div>
      </div>
      <div className="w-full">
        ```bash title="Terminal"
        clarinet new hello-world
        ```
      </div>
      <div className="w-full">
      <p>The result should be a new `hello-world` directory resembling the following:</p>
        <Files className='pointer-events-none'>
          <Folder name="contracts" />
          <Folder name="settings" defaultOpen>
            <File name="Devnet.toml" />
            <File name="Mainnet.toml" />
            <File name="Testnet.toml" />
          </Folder>
          <Folder name="tests" />
          <File name=".gitignore" />
          <File name="Clarinet.toml" />
          <File name="package.json" />
          <File name="tsconfig.json" />
          <File name="vitest.config.js" />
        </Files>
      </div>
    </div>
  </Step>

  <Step>
    <div className="flex flex-col space-y-3">
      <div className="w-full">
        <p className='text-lg font-bold m-0 dark:text-primary-foreground'>Create a <code className='text-lg'>say-hello</code> contract</p>
        <div className='flex flex-col space-y-0'>
          <ol>
            <li>Navigate to your project by running `cd hello-world`.</li>
            <li>Inside your project, use the `clarinet contract new` command to generate a new contract.</li>
          </ol>
        </div>
      </div>
      <div className="w-full">
        ```bash title="Terminal"
        clarinet contract new say-hello
        ```
      </div>
      <div className="w-full">
        <p>This will add 2 files to your project: `say-hello.clar` and `say-hello.test.ts`.</p>
        <Files className='pointer-events-none'>
          <Folder name="contracts" defaultOpen>
            <File name="say-hello.clar" className='bg-muted' />
          </Folder>
          <Folder name="settings" />
          <Folder name="tests" defaultOpen>
            <File name="say-hello.test.ts" className='bg-muted' />
          </Folder>
          <File name=".gitignore" />
          <File name="Clarinet.toml" />
          <File name="package.json" />
          <File name="tsconfig.json" />
          <File name="vitest.config.js" />
        </Files>
        <p>It also updates the `Clarinet.toml` file with your new contract.</p>
        ```toml title="Clarinet.toml"
        [contracts.say-hello]
        path = 'contracts/say-hello.clar'
        clarity_version = 2
        epoch = 2.4
        ```
      </div>
    </div>
  </Step>

  <Step>
    <div className="flex flex-col space-y-3">
      <div className="w-full">
        <p className='text-lg font-bold m-0 dark:text-primary-foreground'>Create a <code className='text-lg'>read-only</code> function called <code className='text-lg'>say-hi</code></p>
        <p>Now that we have our `say-hello.clar` file generated, let's create a `read-only` function that prints out **Hello World**.</p>
      </div>
      <div className="w-full">
        ```clarity title="say-hello.clar"
        (define-read-only (say-hi)
          (print "Hello World")
        )
        ```
      </div>
    </div>
  </Step>

  <Step>
    <div className="flex flex-col space-y-3">
      <div className="w-full">
        <p className='text-lg font-bold m-0 dark:text-primary-foreground'>Verify your contracts</p>
        <p>In order to verify that our code is valid, we can run `clarinet check` inside of our project directory to ensure our `say-hi` function is valid.</p>
      </div>
      <div className="w-full">
        ```bash title="Terminal"
        clarinet check
        âœ” 1 contract checked
        ```
      </div>
    </div>
  </Step>
</div>
