With Clarinet, you can quickly setup your Clarity project through the command line tool. This covers every stage of the development cycle from generating new contracts, an interactive REPL and validating contracts to deployment on a `devnet`, `testnet` or `mainnet`.

## Install Clarinet
<Tabs id="os" items={['macOS', 'Windows']}>
  <Tab value="macOS">
  Run the following command in your terminal:
  
  ```bash
  brew install clarinet
  ```

  {/* For more information on how to install Clarinet on macOS, please see the [Setting Up Your Clarity Development Environment (Mac) video walkthrough](). */}
  </Tab>
  <Tab value="Windows">
  Run the following command in your terminal:

  ```bash
  winget install clarinet
  ```

  {/* For more information on how to install Clarinet on Windows, please see the [Setting Up Your Clarity Environment (Windows) video walkthrough](). */}
  </Tab>
</Tabs>

<Callout title="Additional Installation">For more installation options, check out our [Installation Guide](/docs/clarinet/installation)</Callout>
