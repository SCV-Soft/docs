Let's build a very simple `hello-world` project to demonstrate how easy it is to get started building on Stacks.

<div className='steps space-y-6'>
  <Step>
    <div className="flex justify-between space-x-20">
      <div className="w-1/2">
        <p className='text-lg font-bold m-0'>Install `clarinet`</p>
        <p>First, let's make sure we've installed `clarinet` so that we can use the command line tools.</p>
      </div>
      <div className="w-1/2">
        <Tabs id="os" items={['macOS', 'Windows']}>
          <Tab value="macOS">
          ```bash title="Terminal"
          brew install clarinet
          ```

          </Tab>
          <Tab value="Windows">
          ```bash title="Terminal"
          winget install clarinet
          ```

          </Tab>
        </Tabs>
      </div>
    </div>
  </Step>

  <Step>
    <div className="flex justify-between space-x-20">
      <div className="w-1/2">
        <p className='text-lg font-bold m-0'>Create a new `hello-world` project</p>
        <p>Now let's create a `hello-world` project. This will scaffold all the necessary files and directories for us to get started.</p>
      </div>
      <div className="w-1/2">
        ```bash title="Terminal"
        clarinet new hello-world
        ```
      </div>
    </div>
  </Step>

  <Step>
    <div className="flex justify-between space-x-20">
      <div className="w-1/2">
        <p className='text-lg font-bold m-0'>Create a `say-hello` contract</p>
        <p>Now that our project has been generated we can start creating our first contract. Let's name this one `say-hello`.</p>
      </div>
      <div className="w-1/2">
        ```bash title="Terminal"
        cd hello-world
        clarinet contract new say-hello
        ```
      </div>
    </div>
  </Step>

  <Step>
    <div className="flex justify-between space-x-20">
      <div className="w-1/2">
        <p className='text-lg font-bold m-0'>Create a `read-only` function called `say-hi`</p>
        <p>Now that we have our `say-hello.clar` file generated, let's create a `read-only` function that prints out **"Hello World"**.</p>
      </div>
      <div className="w-1/2">
        ```clarity title="say-hello.clar"
        (define-read-only (say-hi)
          (print "Hello World")
        )
        ```
      </div>
    </div>
  </Step>

  <Step>
    <div className="flex justify-between space-x-20">
      <div className="w-1/2">
        <p className='text-lg font-bold m-0'>Verify your contracts</p>
        <p>In order to verify that our code is valid, we can run `clarinet check` inside of our project directory to ensure our `say-hi` function is valid.</p>
      </div>
      <div className="w-1/2">
        ```bash title="Terminal"
        clarinet check
        âœ” 1 contract checked
        ```
      </div>
    </div>
  </Step>
</div>

And that's it! You can now run `clarinet check` to verify that your contract is valid.
