Clone the [Ordhook repository](https://github.com/hirosystems/ordhook) to your local machine.

```bash title="Terminal"
git clone https://github.com/hirosystems/ordhook.git
```

Navigate to the root directory and run cargo ordhook-install to install:

```bash title="Terminal"
cd ordhook && cargo ordhook-install
```
